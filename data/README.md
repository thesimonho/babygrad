# Datasets

Small CSV datasets for early forward-only network experiments.

## `iris.csv`

Classic Iris flower classification dataset.

- Rows: 150 examples
- Inputs: `sepal_length_cm`, `sepal_width_cm`, `petal_length_cm`, `petal_width_cm`
- Target: `species`
- Classes: `Iris-setosa`, `Iris-versicolor`, `Iris-virginica`
- Source: UCI Machine Learning Repository Iris data file
  (`https://archive.ics.uci.edu/ml/machine-learning-databases/iris/iris.data`)

## Synthetic Target Datasets

Two synthetic datasets for trying the same shared inputs with separate
categorical and continuous targets.

- Rows: 60 examples
- Inputs: `x1`, `x2`, `x3`, `x4`
- Categorical file: `categorical_targets.csv`
- Categorical target: `category_target` (`low`, `medium`, `high`)
- Continuous file: `continuous_targets.csv`
- Continuous target: `continuous_target`
- Generated by: `python data/generate_mixed_targets.py`

The synthetic formulas are intentionally simple and deterministic. They are not
intended as a benchmark; they are just a small dataset for checking data loading,
shape handling, forward passes, and output interpretation.
