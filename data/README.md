# Datasets

Small CSV datasets for early forward-only network experiments.

## `iris.csv`

Classic Iris flower classification dataset.

- Rows: 150 examples
- Inputs: `sepal_length_cm`, `sepal_width_cm`, `petal_length_cm`, `petal_width_cm`
- Target: `species`
- Classes: `Iris-setosa`, `Iris-versicolor`, `Iris-virginica`
- Source: UCI Machine Learning Repository Iris data file
  (`https://archive.ics.uci.edu/ml/machine-learning-databases/iris/iris.data`)

## `concentric_circles.csv`

Synthetic two-class dataset for the Phase 7 residual-network demo. One class is
an inner disk, the other an outer ring, separated by an empty margin. No linear
boundary can separate them, and recovering the boundary requires composing a
radius-like feature (`x1**2 + x2**2`) from the raw inputs — so a shallow network
underfits and a deep *plain* stack struggles to train, while a deep residual
network learns it cleanly.

- Rows: 5000 balanced examples (2500 inner, 2500 outer)
- Inputs: `x1`, `x2`
- Target: `label` (`inner`, `outer`)
- Generated by: `python data/generate_concentric_circles.py`

The generator is deterministic (fixed seed). Radii are area-weighted so points
spread evenly across each ring, with small Gaussian jitter on each coordinate.
Used by `notebooks/phase7_residual_networks.ipynb`, which trains on a slice of
the rows to keep the pure-Python run to a couple of minutes.
